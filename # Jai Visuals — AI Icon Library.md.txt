# Jai Visuals — AI Icon Library

A scalable vector icon library for AI, LLM, neural networks, deep learning, RAG, agents, embeddings, and modern system architecture.

Each icon is designed as a single-path silhouette on a teal artboard, with consistent geometry, spacing, and rhythm across the full collection.

Designed for use in UI dashboards, diagrams, technical documentation, and AI workflow illustrations.

## Structure
- `/ai-core concepts` — core AI/ML concept icons
- `/LLM_Core` — large language model specific concepts
