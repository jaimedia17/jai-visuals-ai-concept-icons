## Concept
Context retrieval in AI models is the process of fetching relevant 
stored information and injecting it into the context window alongside 
the current query to inform generation. Across sessions this process 
evolves — each conversation turn refines the context state carried 
forward, shifting from broad general retrieval toward precise 
personalised retrieval as session history accumulates.

## Visual Interpretation
Vertical pipeline of three repeating sessions reading top to bottom. 
Each session consists of a structured knowledge grid on the left 
feeding into a central chat box with right-aligned query and 
left-aligned response, followed by a context circle on the right. 
Dashed connectors carry context state forward from each circle into 
the next session. The repeating pattern implies an ongoing cycle 
beyond the depicted frame.
